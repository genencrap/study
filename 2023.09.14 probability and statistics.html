probability = 불확실성, 기회를 다루는 과학적인 도구

표본공간 S(sample space) = 모든 가능한 실험 결과들의 집합
S = {(1,1),(1,2)...}
S = [0,inf) / 0이상 inf 미만
시행(trial) = 동일한 조건 아래서 반복 가능
독립 시행(independent trial) = 동일 조건에서 어떤 시행을 반복할때, 각 시행의 결과가 이전 실행의 결과에 영향을 받지 않는 시행
복원추출(replacement) , 비복원 추출(without replacement)
원소(element) or 표본 점(sample point) = 시행에서 나타날 수 있는 개개의 결과
근원사건 = 단 하나의 원소로 구성된 사건
공사건 = 원소가 하나도 들어있지 않은 사건

사건 또는 사상(event) E 는 가능한 한 개 이상의 결과(outcome)들이 나오는 프로세스,절차로서,E는 표본공간 S의 부분집합
사건 E의 확률은 P(E) 로 표기
사건 E의 여집합(complement)E_는 사건E를 포함하고 있지 않는 표본공간 내의 모든 사건들로 구성됨
사건 E와 F의 교집합(intersection)은 EnF로 표기 / P(EnF)는 사건 E와F가 동시에 발생할 확률
두 사건 E와 F가 E n F = 공사건 일때 사건 E 와 F는 상호배반(mutually exclusiv or disjoint)라고 한다
사건 E와 F의 합집합(Union)은 E U F 로 표기
A U (B n C) = (A U B) n (A U C)
표본공간의 분할(partition)은 상호배반인 사건 E1,E2...En의 과정(sequence)로서 E1 U E2 U ...En = S를 만족

공리(Axioms) = 너무 당연한거

- 0 <= P(E) <= 1
  • - P(S) = 1
  • - if E1,E2..가 상호 배반일때 P(U i=1~inf Ei) = sigma i=1~inf P(E)
P(E U F) = P(E) + P(F) - P(E n F)
P(U i=1~n Ei) = sigma i=1~n P(E) - sigma i < j P(Ei n Ej) ... (-1)^(n+1) * P(n i=1~n Ei)

통계적 확률(statistic probability) = 같은 시행을 n번 반복했을때 사건 A가 일어난 횟수를 rn 이라 하면
n이 한없이 커짐에 따라 상대 도수 rn / n 가 가까워지는 일정한 값 p

기하학적 확률(geometrical probability) = 표본공간과 사건 A가 무한집합이어서 그 수를 셈할 수 없는 경우,
기하학적 확률을 적용해 구한다. (사건 A가 일어날 기하학적 확률) = (사건 A가 일어날 영역의 크기) / (일어날 수 있는 전 영역의 크기)

사건 F가 주어졌을때 사건 E의 조건부 확률(conditional probability)

- P(E|F) = P(E n F) / P(F) -> P(E n F) = P(F)P(E|F) , P(F)>0
P(S|F) = 1 , P(n i=1~n E) = P(E1)P(E2|E1)P(E3|E1nE2)...P(En|E1nEn-1)
조건부 확률 P(Ei | E1 n ... n Ei-1)을 쉽게 구할 수 있을때 유용.
사건의 독립(independent)
- P(E n F) = P(E)P(F) => P(E|F) = P(E) 일때 사건 E와 F는 독립

전확률의 법칙(low of total probability)

- E1,E2 .. En의 분할이라 할 때, P(E) = sigma i=1~n P(E n Ei) = sigma i=1~n P(E|Ei)P(Ei)
pairwise independence != independenve => ab, ac, bc 가 각각 독립이여도 abc는 독립이 아닐 수 있음
사후 확률(posterior probability)

베이즈 정리(bayes' theorem)

= 만약 E1, E2.. En이 표본공간의 분할이라고 한다면 사건 F가 주어졌을때 사건 Ei가 발생할 사후확률은
P(Ei|F) = P(Ei)P(F|Ei) / sigma j=1~n P(Ej)P(F|Ej)
전체 n개로부터 k개를 선택하는 가지 수 = n^k
비복원추출로 전체 n개로부터 k개를 선택하는 가지 수는? = n(n-1)...(n-k+1)

순열(permutation)

=> n개로부터 k를 선택하는 가능한 순열의 수는 n! / (n-k)!
만약 k=n이면 순열의 수는 n!/(n-n)! = n!
순열(permutation) = 선택의 순서가 중요함

조합(combination)

= 선택의 순서가 중요하지 않음
n개로부터 k(n>=k)개를 뽑는 조합(combination)은 비복원추출로 n개로부터 k개를 순서에 관계없이 선택하는 경우의 수
가능한 조합의 수는 = n!/(n-k)!k!
C n,k = C n,n-k
